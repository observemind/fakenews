Add a `Procfile`:


### Option C: VPS / DigitalOcean

```bash
# Server setup
sudo apt update && sudo apt install python3-pip nginx -y

# App setup
pip install -r requirements.txt
python train_model.py
gunicorn app:app --bind 0.0.0.0:5000 --workers 4 --daemon

# Nginx reverse proxy
sudo nano /etc/nginx/sites-available/satyacheck
# Add: proxy_pass http://127.0.0.1:5000;
```

### Option D: Docker

```dockerfile

WORKDIR /app
COPY . .
RUN pip install -r requirements.txt
RUN python train_model.py
EXPOSE 5000
CMD ["gunicorn", "app:app", "--bind", "0.0.0.0:5000"]
```

